# 🎓 同学城市分布地图 - 升级版

基于 Python Flask + ECharts 的交互式可视化工具，适合大学/高中同学群统计城市分布。

## ✨ 功能特点

- 🗺️ **中国地图可视化** - 直观展示同学们在全国的分布
- 📊 **城市排行榜** - 实时统计各城市人数 TOP 榜
- 👥 **同学列表** - 点击查看每位同学详情
- ➕ **在线添加** - 无需编辑 Excel，网页直接添加
- 📥 **数据导出** - 一键导出 Excel 文件
- 🎨 **炫酷界面** - 渐变配色 + 动画效果
- 📱 **响应式设计** - 手机/平板/电脑都能用

## 🚀 快速开始

### 1. 安装依赖

```bash
cd classmate_map
pip3 install -r requirements.txt
```

或使用国内镜像：
```bash
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 启动应用

**方法一：直接运行**
```bash
python3 app.py
```

**方法二：使用启动脚本**
```bash
chmod +x start.sh
./start.sh
```

### 3. 访问应用

- **主页面（地图）**: http://localhost:5000
- **管理页面**: http://localhost:5000/admin

## 📖 使用指南

### 添加同学信息

1. 点击页面右下角的 **"+"** 按钮
2. 填写姓名、城市（必填）和微信号、职业、备注（选填）
3. 点击"添加"按钮
4. 地图和排行榜会自动更新

### 查看城市分布

- **地图模式**：鼠标悬停查看城市人数，滚轮缩放地图
- **排行榜**：右侧显示城市人数排名
- **同学列表**：点击同学头像查看详细信息

### 管理数据

访问 http://localhost:5000/admin 可以：
- 查看所有同学信息表格
- 导出数据为 Excel 文件
- （删除功能需在后端实现）

### 分享给同学

**方案 1：局域网访问**
1. 确保手机和电脑在同一 WiFi
2. 查看电脑 IP 地址（如：192.168.1.100）
3. 同学访问 http://192.168.1.100:5000

**方案 2：部署到公网**
- 使用云服务器（阿里云/腾讯云）
- 使用内网穿透工具（ngrok/frp）
- 部署到 Heroku/Vercel 等平台

## 🔧 高级配置

### 修改端口

编辑 `app.py` 最后一行：
```python
app.run(debug=True, port=8080, host='0.0.0.0')  # 改为 8080 端口
```

### 添加更多城市坐标

编辑 `city_coordinates.json`，添加格式：
```json
{
  "城市名": [经度, 纬度]
}
```

### 自定义界面样式

编辑 `templates/index.html` 中的 CSS 部分，修改颜色、布局等。

## 🐛 常见问题

### 1. 地图不显示

**原因**：ECharts 中国地图数据加载失败

**解决方法**：
- 确保网络畅通（需要加载 CDN 资源）
- 或下载 `china.js` 到本地，修改 `index.html` 中的引用

### 2. 城市坐标不匹配

**原因**：输入的城市名不在坐标库中

**解决方法**：
- 编辑 `city_coordinates.json`，添加该城市坐标
- 或输入坐标库中已有的城市名（如"北京市"而非"北京"）

### 3. 端口被占用

**错误信息**：`Address already in use`

**解决方法**：
```bash
# 查看占用端口的进程
lsof -i :5000

# 杀死进程
kill -9 <PID>

# 或修改 app.py 中的端口号
```

### 4. 数据丢失

**原因**：`data/classmates.xlsx` 被删除

**解决方法**：
- 从导出的 Excel 备份恢复
- 或重新添加数据

## 📊 数据格式

`data/classmates.xlsx` 包含以下字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 姓名 | 文本 | ✅ | 同学姓名或昵称 |
| 城市 | 文本 | ✅ | 所在城市（如：北京、上海） |
| 微信 | 文本 | ❌ | 微信号（选填） |
| 职业 | 文本 | ❌ | 职业（选填） |
| 备注 | 文本 | ❌ | 备注信息（选填） |
